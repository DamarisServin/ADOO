Casos de Uso

CU05. Subir Canción

Identificador del CU: CU05 

Nombre: Subir canción. 

Descripción: Se ingresa una nueva canción al sistema. 

Actores: Conductor de radio. 

Precondición: La canción debe tener un nombre descriptivo. 

Postcondición: Se agrega una nueva canción al sistema.

Entradas: Canción. 

Salidas: Se muestra en pantalla que la canción ha sido agregada con éxito y se envía una notificación de “cambio en espera” a la estación de radio. 

Trayectoria principal: 

  1. El conductor de radio ingresa a la página web.
  2. El conductor de radio sube la canción [Trayectoria Alt. A: La canción no está en formato .mp3 o equivalente].
  3. El sistema valida que la canción no exista en el sistema [Trayectoria Alt. B: La canción ya existe en el sistema].
  4. El sistema muestra un mensaje de que la canción ha sido agregada al sistema con exito.
  5. Se envía una notificación a la estación de radio de que hay nueva información en espera de ser revisada.
  6. Termina el caso de uso. 

Trayectoria Alternativa A: 

          
    A.3 El sistema muestra en pantalla que el tipo de archivo es incorrecto de acuerdo a la BR4.
    A.4 El sistema despliega una lista de los formatos permitidos.
          
    A.5 Termina el caso de uso.

Trayectoria Alternativa B:

    B.4 El sistema despliega el mensaje “La canción ya existe en el sistema”.
    B.5 Termina el caso de uso.
