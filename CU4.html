Casos de Uso

CU04 Subir Álbum

Identificador del CU:  CU04 

Nombre:  Subir álbum. 

Descripción:  Se ingresa un nuevo álbum al sistema. 

Actores:  Conductor de radio. 

Precondición:  Cada canción debe tener un nombre y el álbum debe ser un solo archivo. 

Postcondición:  Se agrega un nuevo álbum a la base de datos del sistema en espera de ser aceptado por la estación de radio.

Entradas:  Archivo con canciones. 

Salidas:  Se muestra en pantalla que el álbum ha sido cargado con éxito y se envía una notificación de “cambio en espera” a la estación de radio. 

Trayectoria principal:

  1. El conductor de radio ingresa a la página web
  2. El conductor de radio sube el archivo con las canciones. 
  3. El sistema valida que el archivo no este vacío [Trayectoria Alt. A: El archivo está vacío].
  4. El sistema valida que el formato de las canciones sea correcto [Trayectoria Alt. B: El formato de las canciones no es .mp3 o equivalente] de acuerdo a la regla de negocio BR4.
  5. El sistema valida que el álbum no exista en el sistema [Trayectoria Alt. C: El álbum ya existe en el sistema].
  6. El sistema muestra un mensaje de que el álbum ha sido agregado al sistema.
  7. Se envía una notificación a la estación de radio de que hay nueva información en espera de ser revisada.
  8. Termina el caso de uso. 

Trayectoria Alternativa A:

          
    A.4 El sistema informa que el archivo está vacío.
          
    A.5 Termina el caso de uso.

Trayectoria Alternativa B:

    B.5 El sistema muestra en pantalla un mensaje “El formato de las canciones es incorrecto”.
    B.6 El sistema muestra un listado de formatos aceptados.
    B.7 Termina el caso de uso.

Trayectoria Alternativa C:

    C.6 El sistema despliega el mensaje “El álbum ya existe en el sistema”.
    C.7 Termina el caso de uso.
